# 📚 Online Learning Portal (TutorTrek)

A full-featured online learning platform that connects tutors and students in a structured, course-based environment.

## 🚀 Features

- Course creation and management by instructors
- Student enrollment and progress tracking
- Structured lesson modules with content delivery
- User authentication and role-based access (student / instructor)
- Responsive UI for seamless learning experience

## ⚙️ Setup & Run

1. Clone the repository
```bash
   git clone https://github.com/sainath1308/online-learning-portal.git
```
2. Import the database schema from `db/schema.sql` into MySQL
3. Update DB credentials in the config file
4. Run the application
